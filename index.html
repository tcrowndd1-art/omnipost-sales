📌 "독립 블로그"란?

OmniPost 1개 = 7개 언어가 하나의 도메인에 자동 발행됩니다 (예: myblog.com/en, /pt, /ko…). Pro/Agency는 완전히 분리된 블로그를 여러 개 운영 — 주제별, 클라이언트별, 또는 국가별 독립 도메인으로 브랜딩할 때 사용합니다.

📦 구매 후 어떻게 받나요?

결제 즉시 Gumroad가 이메일로 전달합니다 — ① 소스코드 zip ② GitHub 비공개 저장소 초대 ③ 셋업 가이드. 로그인 없이 구매 가능하며, 언제든 다시 다운로드할 수 있습니다.

Q진짜 효과가 있나요? AI 글을 사람이 읽을까요?
이 전략을 쓰는 회사가 HubSpot, Airbnb, NerdWallet입니다. 구글 공식 입장도 "독자에게 유용하면 AI 글도 괜찮다"입니다. OmniPost는 리서처가 최신 정보를 수집해 실질적 글을 씁니다.
Q내 사업 설명은 어디에 넣나요?
다운로드한 폴더 안 docs/ 폴더에 메모처럼 사업 설명 파일 하나를 넣으면 됩니다. 코딩도, 정해진 서식도 없습니다.
Q발행 시간과 횟수를 정할 수 있나요?
네. 원하는 시간(예: 매일 9·15·21시)과 건수를 직접 설정합니다. 하루 3건은 권장값일 뿐 고정이 아닙니다.
QAI 비용은 얼마나 드나요? 무료도 되나요?
Gemini Flash 기준 글 1편(7개 언어)이 약 $0.05~0.15입니다. 그리고 이미 Claude Pro·ChatGPT Plus를 쓰고 있다면 CLI 모드로 추가 비용 $0입니다.
Q개발자가 아닌데 설치가 어렵지 않나요?
GitHub·Vercel 계정은 무료이고 가이드에 스크린샷이 있습니다. 템플릿 선택, 컬러 선택, 사업 설명 작성 — 평균 5분이면 첫 발행이 시작됩니다.
Q환불은 어떻게 되나요?
구매 후 7일 이내 전액 환불됩니다. 소스코드 zip은 회수되지 않지만, GitHub 저장소 접근은 환불과 함께 회수됩니다.
